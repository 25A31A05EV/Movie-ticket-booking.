# Ticket Booking Program

This Python program simulates a **movie ticket booking system**. It checks user eligibility based on age and show time, calculates **discounts, extra charges, and service charges** based on membership, weekend shows, and seat type.

## Features
- Checks if the user is eligible to book a ticket.  
- Verifies eligibility for evening shows.  
- Applies membership discounts if applicable.  
- Adds extra charges for weekend or evening shows.  
- Calculates service charges based on seat type (Premium, Gold, or others).  
- Prints a detailed summary including eligibility, discounts, extra charges, and service charges.

## How to Use
1. Open `movie_ticket_booking.py`.  
2. Update the input variables: `age`, `seat_type`, `show_time`, `is_member`, `is_weekend`.  
3. Run the program using Python 3:
```bash
python movie_ticket_booking.py
